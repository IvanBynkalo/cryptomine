## 🎯 Главные улучшения

### 1. Иконки категорий

```
БЫЛО:                   СТАЛО:
🚀 Корпуса      →      🛸 Корпуса
⚔️  Оружие      →      🔫 Оружие
🔬 Спецмодули   →      ⚙️ Спецмодули
```

**Почему изменили:**
- 🛸 лучше показывает космический корабль
- 🔫 четко показывает оружие (⚔️ - это меч, не космическое оружие)
- ⚙️ универсальный символ для технических модулей

---

### 2. Названия уровней (Tier)

```
БЫЛО:          СТАЛО:
┌─────────┐   ┌──────────────┐
│  Все    │   │  Все         │
│  T1     │   │  Лёгкие      │
│  T2     │   │  Средние     │
│  T3     │   │  Тяжёлые     │
│  T4     │   │  Военные     │
│  T5     │   │  Экспер.     │
│  T6     │   │  Легенд.     │
└─────────┘   └──────────────┘
```

**Почему изменили:**
- Игрок сразу понимает, что означает каждый уровень
- Не нужно запоминать T1 = лёгкие, T2 = средние и т.д.
- Более интуитивный интерфейс

---

### 3. Иконки предметов

#### Корпуса кораблей:

```
Лёгкие истребители:
БЫЛО: 🚀 Swift-M
СТАЛО: 🛩️ Swift-M
Почему: 🛩️ больше похож на лёгкий быстрый корабль

Боевые корабли:
БЫЛО: 🔴 Fighter MK-II
СТАЛО: 🚁 Fighter MK-II
Почему: 🚁 показывает маневренность боевого корабля

Тяжёлые корабли:
БЫЛО: 💪 Bastion-X
СТАЛО: 🚂 Bastion-X
Почему: 🚂 показывает массивность и мощь
```

#### Оружие:

```
Баллистика (Railgun):
БЫЛО: 💥 Railgun Mk-I
СТАЛО: 🔫 Railgun Mk-I
Почему: 🔫 четко показывает оружие

БЫЛО: 🔫 Railgun Mk-II
СТАЛО: ⚡ Railgun Mk-II
Почему: ⚡ показывает энергетическую природу рейлгана

БЫЛО: ⚡ Railgun Mk-III
СТАЛО: 💥 Railgun Mk-III
Почему: 💥 показывает разрушительную мощь
```

## 🔄 Как это работает

### Переключение между категориями

```
┌────────────────────────────────────────────┐
│  📋 КАТАЛОГ ТЕХНИКИ (180 ЕД.)             │
├────────────────────────────────────────────┤
│  [🛸 Корпуса 24]  [🔫 Оружие 54]         │
│  [🛡️ Защита 24]   [🔥 Двигатели 24]      │
│  [⚙️ Спецмодули 36] [🤖 Дроны 18]        │
└────────────────────────────────────────────┘

Клик на любую кнопку → переключается категория
```

### Фильтр по уровню

```
┌────────────────────────────────────────────┐
│  [Все] [Лёгкие] [Средние] [Тяжёлые]      │
│  [Военные] [Экспер.] [Легенд.]            │
│                         [✅ Доступные]     │
└────────────────────────────────────────────┘

Клик на "Лёгкие" → показывает только T1
Клик на "Все" → показывает всё оборудование
Клик на "✅ Доступные" → только то, что можно купить
```

## 🎮 Пример использования

### Игрок хочет найти тяжёлый боевой корабль:

**Шаг 1:** Открыть "Ещё" → "Каталог"

**Шаг 2:** Нажать на 🛸 Корпуса

**Шаг 3:** Нажать на "Тяжёлые" (это T3)

**Шаг 4:** Просмотреть доступные корабли:
- 🚂 Bastion-X (боевой)
- 🛡️ Ranger Cruiser (универсал)
- 🌌 Horizon Deep (исследователь)
- 🏗️ Atlas Heavy (грузовой)

**Результат:** Игрок сразу понимает:
- Какой тип корабля (по иконке)
- Какой уровень (Тяжёлые вместо T3)
- Какая специализация (по описанию)
